# XpenserThis is a repo for managing machine readable artifacts for the [Xpenser APIs](http://xpenser.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Xpenser APIs](http://xpenser.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://xpenser.com/)- [Developer](http://xpenser.com/)- [Twitter](https://twitter.com/#!/xpn)The [APIs.json](https://github.com/api-evangelist/xpenser/blob/master/apis.json) index provides links to all the relevant links for the Xpenser APIs, including the OpenAPI and Postman if they are present.
